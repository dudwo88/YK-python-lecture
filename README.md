# 🐍 Python Basics with My Kid

A structured 12-session Python curriculum designed for a parent and child to learn together — building real mini-projects every session.

> Comparisons to Java are included throughout, since the kid already has some Java background.

---

## 📚 Curriculum Overview

| Session | Topic | Content | Assignment |
|---------|-------|---------|------------|
| S01 | Setup + Hello World | Install Python/VS Code, run first `.py` file, `print()`, comments | **Guess the Number v0** — use only `print()` to display the "game" |
| S02 | Variables & Data Types | `int`, `float`, `str`, `bool`, `type()`, naming rules, compare with Java | **Temperature Converter** — input Celsius, output Fahrenheit |
| S03 | Lists | Create/modify/slice lists, `len()`, `append()`, compare with `ArrayList` | **Favourite Foods list** — reverse, sort, search, add an item |
| S04 | Operators & Strings | Arithmetic/comparison/logical operators, f-strings, string methods | **BMI Calculator** — input height & weight, output value + category |
| S05 | Conditions | `if`/`elif`/`else`, nesting, indentation vs Java braces | **Grade Calculator** — input score, output letter grade A–F |
| S06 | Loops | `for`/`while`, `range()`, `break`/`continue` | **Times Tables (2–9)** + **FizzBuzz (1–100)** |
| S07 | Functions — Basics | `def`, parameters, `return`, default args, docstrings | **Calculator** — four functions: add, subtract, multiply, divide |
| S08 | Functions — Advanced + Modules | Multiple returns, `*args`, `import random` & `math` | **Guess the Number v1** — random target, hints, attempt counter |
| S09 | Dictionaries | Create/read/update/delete, `keys()`/`values()`/`items()`, compare with `HashMap` | **Vocabulary Quiz** — word-definition dict + random quiz mode |
| S10 | Classes & Objects | `__init__`, `self`, instance methods, compare with Java constructors | **Animal class** + Dog, Cat, Cow objects with `speak()` |
| S11 | File I/O | `open`/read/write/append, `with` statement | **Text Diary** — write/read entries via a menu interface |
| S12 | Exception Handling + Review | `try`/`except`/`finally`, common error types, Phase 1 recap | **Robust Calculator** — handle bad input, division by zero |

---

Each folder contains the session's exercises and the completed assignment.

---

## 🛠️ Requirements

- Python 3.x
- VS Code (recommended)

---

## 🎯 Goals

- Learn Python fundamentals through hands-on projects
- Build something fun and working every single session
- Bridge concepts from Java to Python where helpful
- Have a great time coding together 🙌
